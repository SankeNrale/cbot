# cbot
A web implementation of a chatbot

Local Setup:
                  
                  
       Ensure that Python, Flask, SQLAlchemy, and ChatterBot are installed
       (either manually, or run pip install -r requirements.txt).
                            
Run app.py with 
       
       python app.py.
       
The demo will be live at 
       
    http://localhost:5000/ 
       
or       
    
    http://127.0.0.1:5000/
